# Vision

## Vision Statement

Agent UI Workshop is a fullstack chatbot starter kit designed to let developers get a working AI agent — complete with streaming chat and a live data panel — running in under an hour. By removing scaffolding friction, the project lets workshop participants stay focused on what matters: designing agent logic, crafting prompts, and building the UI surfaces that make AI interactions useful.

## Problem Statement

Building an AI chatbot from scratch requires wiring together many layers: an LLM provider, a streaming protocol, a React UI, a backend server, and typed data contracts between them. For developers new to this space, that setup cost dominates workshop time and obscures the core concepts. Most starter templates either oversimplify (hiding the important parts) or over-engineer (drowning beginners in abstraction).

Agent UI Workshop solves this by shipping a production-quality but deliberately readable codebase where every layer is visible and well-separated — but none of it needs to be built from scratch.

## Target Audience

| Persona               | Description                                                             | Key Needs                                                            |
| --------------------- | ----------------------------------------------------------------------- | -------------------------------------------------------------------- |
| Workshop participant  | Developer attending a structured AI workshop (beginner to intermediate) | Clear entry points, commented architecture, quick wins               |
| Self-directed learner | Developer exploring AI SDK and Pydantic AI independently                | Working examples, easy to extend, understandable code                |
| Prototype builder     | Developer starting a new AI product and wanting a solid base            | Production-quality patterns, real streaming, easily swappable agents |
